# CS495-Deepfake-Guardian

## Project Title
CS495-Deepfake-Guardian

## Project Description
This project focuses on real-time deepfake recognition using live or recorded video. The goal is to explore machine learning and computer vision methods that can identify manipulated facial or video content and help improve trust, safety, and authenticity in digital media.

## Objectives
- Build a system for detecting deepfake content in video
- Explore machine learning and computer vision methods for classification
- Evaluate model accuracy and performance on deepfake datasets
- Investigate whether detection can be done in near real-time
- Present meaningful findings related to digital media safety and trust
